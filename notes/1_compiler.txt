
    คอมพิวเตอร์จะเข้าใจแค่ machine code เราจึงต้องใช้ Compiler ทำการแปล Source code ของเรา
    ที่มนุษย์อ่านเข้าใจ ไปเป็นภาษาเครื่องเพื่อให้คอมพิวเตอร์เข้าใจ Source code ที่เราเขียน

    * ตัวอย่าง Machine code: 00000 10011110

    * ตัวอย่าง c++ code:
        int a, b, sum;
     
        cin >> a;
        cin >> b;
             
        sum = a + b;
        cout << sum << endl;

    * c++ ถูกออกแบบให้เป็นภาษา compiler ซึ่งทำให้เครื่องสามารถเข้าใจได้โดยตรง ทำให้โปรแกรมที่ถูกสร้างขึ้น
    มีประสิทธิภาพสูง

    * โดยจำเป็นต้องมีเครื่องมือที่เรียกว่า Toolchain และ Linker

    * Console Program: เป็นโปรแกรมที่ใช้ข้อความสื่อสารกับผู้ใช้และ environment เช่น การพิมพ์ข้อความบนจอภาพ
