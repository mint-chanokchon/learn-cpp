
    * << >> บอกทิศทางของการ stream data

    * cout << (insertion operator)
        - stream ข้อมูลไปที่ cout (แสดง output ไปที่หน้าจอ)

    * cin >> (extraction operator)
        - stream ข้อมูลที่ได้จาก input ไปใส่ตัวแปรที่ระบุ

    * ถ้าหากต้องการรับ input ที่มี่องว่างได้ ให้ใช้ getline 
        - อาจะมีปัญหาตามมา เพราะจะทำให้มีค่าค้างอยู่ในบัฟเฟอร์
        - ใช้ std::ws (white space) เพื่อทำการลบบรรทัดใหม่ อักขระ หรือช่องว่างใด ๆ ก่อนหน้านี้ที่ผู้ใช้ป้อนเข้ามา
