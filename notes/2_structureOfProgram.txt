    // my first program in C++
    #include <iostream>

    int main()
    {
        std::cout << "Hello World!";
    }

    * // my first program in C++
        - บรรทัดนี้คือการคอมเมนท์ เมื่อถูก compile จะไม่ถูกอ่าน

    * #include <iostream>
        - บรรทัดที่ขึ้นต้นด้วย # คือ คำสั่งที่อ่านและตีความโดย preprocessor โดยจะถูกตีความก่อนจะเริ่ม compile
        - โดยในตัวอย่าง จะเป็นการสั่งให้ประมวลผลล่วงหน้าในส่วนของ standard c++ ที่มีชื่อว่า iostream ซึ่งทำให้สามารถดำเนินการ input, output
        เช่น การเขียนโปรแกรม Hello World 

    * int main()
        - เป็นการเริ่มต้นประกาศฟังก์ชัน โดยจะเป็นกลุ่มของคำสั่งขแง source code ที่ได้รับการตั้งชื่อ โดยในกรณีนี้จะชื่อว่า main

    * { /* code */ }
        - เป็นชุดคำสั่งที่จะถูกเรียกใช้งานเมื่อฟังก์ชันนั้นถูกเรียกใช้งาน

    * std::cout << "Hello World!";
        - นิพจน์ คือการนำตัวแปรหรือค่าคงที่มาดำเนินการกันด้วยเครื่องหมายทางคณิต
        - เป็นการระบุพฤติกรรมของโปรแกรม
        > โดยด้านในคำสั่งนี้จะมีทั้งหมด 3 ส่วน
            - 1 std::cout เป็นการเรียกใช้ namespace standard โดยเราจะเข้าถึง cout หรือ Character output device โดยปกติคือหน้าจอคอมพิวเรอตี์
            - 2 << หรือ insertion operator ระบุว่าสิ่งต่อไปนี้จะถูกสร้างลงไปใน std::cout
            - 3 "Hello World" คือ เนิ้อหาที่เราทำการ insert ลงไปใน standard output

    * ข้อสังเกตุ: ในภาษา c++ ชุดคำสั่งจะลงท้ายด้วย ; เสมอ เพราะ ; จะเป็นตัวที่ทำหน้าที่บอกว่า ถึงจุดสิ้นสุดของคำสั่ง เช่นเดียวกับในภาษาอังกฤษที่ประโยค
    เมื่อจบแล้วจะต้องลงท้านด้วยจุด (.)

    * การแยกระหว่างคำสั่ง จะใช้เครื่องหมายอัฒภาค (;)

    * คำสั่งที่ถูกประมวลผลล่างหน้า (ขึ้นต้นด้วย # หรือ preprocessor) อยู่นอกเหนือกฏทั่วไป เนื่องจากไม่ใช่คำสั่ง เป็นบรรทัดที่อ่านและประมวลผลล่วงหน้าก่อน
    ที่จะเริ่ม compile ที่เหมาะสม และเนื่องจากไม่ใช่คำสั่ง จึงไม่จำเป็นต้องลงท้ายด้วย ;

    * Using namespace std
        - cout เป็นส่วนหนึ่งของ standard library โดยเราสามารถประกาศเรียกใช้ library ไว้ด้านบนได้ โดนการ using namespace
        - using namespace std; ทำให้เราสามารถเข้าถึงองค์ประกอบทั้งหมดของ library ได้แบบไม่เหมาะสม
