
    Dynamic Memory

    * ใน c++ เราสามารถจัดการกับหน่วยความจำได้ในเวลาที่โปรแกรมทำงาน 
    เช่น การจองหน่วยความจำ, การคืนหน่วยความจำ

    * ในการจัดการ Dynamic Memory เราจะใช้ keyword 'new' สำหรับตัวแปรปกติ และ 'new[]' สำหรับ array
    โดยมันจะส่งคืนตำแหน่งเริ่มต้นใน Memory คืนมา

    * example:
        int *numbers;
        numbers = new int[5];
        - ในตัวอย่าง เราต้องการจัดสรรหน่วยความจำแบบไดนามิกสำหรับตัวแปร integer 5 ตัวแปร
        - เราประกาศ numbers ขึ้นมา ซึ่งประเภทของมันจะต้องตรงกับค่าของข้อมูลที่เราจะจัดสรรค์หน่วยความจำให้
        - บรรทัดต่อมา เราใช้คำสั่ง new เพื่อจัดสรรหน่วยความจำที่มีขนาด 5 และโดยระบุขนาดไว้ด้านใน [] 
        ซึ่งต้องระบุเป็นจำนวนเต็มบวก
        - หลังจากนั้น Pointer จะชี้ไปยังตำแหน่งแรกของหน่วยความจำที่จองไว้

    * Operators delete and delete[]
        - โดยปกติโปรแกมจะทำการคืนหน่วยความจำให้ระบบหลังจากโปรแกรมสิ้นสุดการทำงานอย่างสมบูรณ์
        แต่บางครั้งเราได้เสร็จสิ้นงานของเราแต่โปรแกรมยังทำงานไม่จบ เราสามารถคืนหน่วยความจำได้โดยใช้ delete

        example:
            delete numbers;     // ส่งคืนหน่วยความจำสำหรับตัวแปร
            delete[] numbers;   // ส่งคืนหน่วยความจำสำหรับ array

    * Dynamic memory vs array;
        - array เมื่อทำการประกาศขนาดแล้วจะทำการจองพื้นที่ใน memory ไว้ทันที
        - dynamic memory การจองพื้นที่จะขึ้นอยู่กับจำนวนข้อมูล
